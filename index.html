The Seed Project

Don't mind this horrible page, i will certainly make a beautiful one when I get the creativity.

What is it?

This is a project that aims to create a virtual reality platform, much like what (spoiler ahead) Kayaba Akihiko gives to Kirito through the Seed Of The World (that's where the name comes from).

The project will contain some subprojects: for now, I'm thinking on subprojects for the engine itself, a sample game using it (something like ALO), and the swarm controller app (more details on it later).

How?

This is an open source project, so code from everyone will be welcome (if it actually adds something to the project).

The engine itself will be written in C++. It should be multiplatform, so our graphical backend will be OpenGL (and possibly Vulkan in the future).

The swarm controller will be written in JavaScript using Node.js and the game will be in Lua, as the engine will use it as scripting language.

While most people doesn't own a VR headset, there'll be some systems to provide Cardboard-esque support.
